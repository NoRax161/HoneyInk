# HoneyInk
 Raspberry Pi Zero (2) W H + Waveshare e-Paper HAT + Cowrie. No backlight, no cloud dashboard, no browser tab just a paper-like screen quietly logging every bot that tries to break in, updated in near real time.

## (Inspiration)

Loosely inspired by [Pwnagotchi](https://pwnagotchi.ai/) — same hardware idea (Pi Zero + e-ink), but offensive, collecting WiFi handshakes. HoneyInk is the defensive counterpart: instead of going out and grabbing data, it sits still and waits for attackers to come to it.
